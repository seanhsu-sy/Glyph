# Glyph Release Notes

## 2.0.0 — 2026-04-16

主版本号对齐：**2.0.0**。包含此前 0.1.x 路线中的参照分屏、统计与书库改动，以及 **伏笔系统**（`.glyph/foreshadowing.json`）、**书库创建** 对 Tauri IPC 的检测修复、**构建脚本**（避免 `npm run build dev` 误传参数）、`write_file_ensuring_parent` 等。详见下方历史条目与提交记录。

### macOS 下载后若无法打开（已隔离 / 来自互联网）

将应用拖入「应用程序」后，在 **终端** 执行（路径请按实际安装位置修改）：

```bash
xattr -cr /Applications/Glyph.app
```

若仍提示来自身份不明开发者，可再试清除隔离标记：

```bash
xattr -dr com.apple.quarantine /Applications/Glyph.app
```

也可在 **系统设置 → 隐私与安全性** 中点击 **仍要打开**，或对 `Glyph.app` **右键 → 打开** 完成首次授权。

## 0.1.1 — 2026-04-06

### 新增

- **参照（原侧栏「大纲」）**：工具条与侧栏入口改为 **「参照」**；内分 **大纲 / 章节** 两个标签。在 **章节** 中点选文档，会在主编辑区右侧打开 **参照分屏**，便于对照写作；参照栏内容会 **计入今日字数与写作统计**（与主文档同样写入写作日志）。
- **书籍库**：筛选行右侧 **「+」** 在已有书籍时也可新建书籍；空书架时「新建书籍」为独立按钮。顶栏 **主题 / 音效** 固定在滚动区域顶部。

### 改进

- **码字目标**：统计页在「全部」下保存的全局目标，与写作页底部 **今日 x/目标** 一致（本书目标优先，否则回落到全局，默认 2000）。
- **连续天数与写作日**：只要当日产生写作会话（含删改后净字数为负），即计入 **活跃天与连续天数**；总览中的「写作天数」与日均也与该口径一致。
- **新建书籍与 Untitled**：仅在本机书库 **新建书籍成功** 后、**第一次**进入该书写作区时，才自动生成内存 **Untitled.md**（含首次欢迎文案）；从书库打开 **已有旧书** 不再因空标签而自动生成。
- **编辑器**：修复 **当前行高亮** 重复注册扩展导致的多行异常高亮。

### 移除

- **章节分屏 / 副屏跨书**：已移除双编辑区、拖拽打开第二本书等相关逻辑，避免与「参照」分屏重复。

### 技术

- 前端：`npm run build` 通过；新增 `referencePaneStore`、共享 `resolveWritingTarget`、`getPendingInitialUntitledKey` 等。
- 后端：`stats` 中连续天与总览「写作日」统计口径更新。

## 0.1.0 — 2026-04-04

### 新增

- **批注与便签**：写作工具栏可给选中文本添加批注（正文内暖色高亮）、添加右下角便签；数据存于本机 `localStorage`。
- **跨书关联**：批注/便签可勾选「跨书」；在书籍库顶栏或侧栏「关联」中打开 **跨书关联** 页统一查看与删除。
- **侧栏「关联」**：合并 **标签互文**（`#标签` 与正文关键词）与 **本书批注与便签** 列表，并链到跨书页。
- **用法说明**：见仓库内 `docs/associations.md`。

### 改进

- **编辑器选区与批注颜色**：全局选区由偏蓝的强调色改为**中性灰**；批注范围为**琥珀/暖棕**系，与标签蓝色、统计高亮区分；当前行背景改为浅灰而非蓝调。
- **书籍库 / 统计页**：修复在窗口内 **无法向下滚动** 的问题（flex + `overflow` 布局）。
- **未命名 Untitled.md**：进入写作页时自动打开内存中的 Untitled（首次含欢迎说明）；**保存 / ⌘S** 通过后端命令直接写入**当前书籍目录**下的 `Untitled.md`（占用则 `Untitled-2.md` …），不依赖易阻塞的系统另存对话框；顶栏另存为仍可导出到任意路径。
- **工具栏**：批注/便签按钮默认与周围样式一致（不再默认蓝底）。
- **侧栏关联**：默认收起长说明，可点「显示说明」；批注卡片合并重复的「引用 / 正文」展示。
- **书籍库**：移除「跨书关联」按钮（写作页侧栏「关联」内仍可打开跨书页）。

### 技术

- 前端：`npm run build`（`tsc` + `vite build`）通过。
- 桌面：`cargo check`（Tauri 2）通过。
- 平台：macOS / Windows / Linux 构建以 `src-tauri/tauri.conf.json` 为准。

### 已知限制

- 批注锚点为**字符偏移**，大段删改正文后高亮可能错位，可删后重建。
- 关联数据仅保存在本机浏览器存储，**未做云同步**；清除站点数据会丢失。

### 升级说明

自本版本起首次使用批注/便签会自动写入 `localStorage` 键 `glyph_associations_v1`，与既有书籍、统计数据互不覆盖。
