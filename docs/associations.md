# 批注、便签与「关联」面板

数据保存在本机浏览器的 `localStorage`（键名 `glyph_associations_v1`），仅当前设备可见。

## 未命名文稿（Untitled.md）

进入某本书的写作页时，若没有打开任何章节，会自动出现 **Untitled.md**（内存占位）。**本机第一次**在任意书中自动打开该占位时，会预填操作说明（`DEFAULT_WELCOME_MARKDOWN`）；之后（含新开的书）同一浏览器内自动打开则为空白草稿。说明文案见 `src/shared/lib/welcomeContent.ts`，是否已展示由 `localStorage` 键 `glyph_welcome_seen_global_v1` 记录。

- **保存 / ⌘S**：将内容**直接写入当前书籍文件夹**为 `Untitled.md`（若已存在则 `Untitled-2.md` …），**不弹系统对话框**，保存后左侧文档列表会刷新。
- **另存为**：仍可用顶栏「另存为」将内容导出到**任意路径**（系统对话框）。
- 从内存草稿保存到真实路径后，本书内针对该草稿的批注会**自动跟到新路径**。

侧栏「关联」右上角可 **显示说明 / 隐藏说明**，需要时再展开长文案。

跨书关联仅从**写作页**侧栏「关联」底部进入（或相关入口），书籍库首页不再提供。

## 工具栏

| 按钮 | 作用 |
|------|------|
| **批注** | 先在正文里**拖选一段文字**，再点「批注」，在弹出条里写说明。可选「跨书关联」：勾选后，该条会进入**跨书关联**页，便于在多本书之间对照。 |
| **便签** | 在屏幕**右下角**出现半透明便签，随手记。便签上可勾选 **跨书**，同样会出现在跨书关联页。 |

批注在编辑器里会以**暖色背景**标出选区；便签点 **×** 仅为隐藏，可在侧栏列表里删除彻底移除。

## 侧栏「关联」（写作页右侧工具）

打开侧栏后点 **关联**，同一面板内分为两块：

1. **标签互文**  
   根据你在文中使用的 `#标签`，在正文里查找**同名关键词**（跳过标签自身），用于设定/人物等互文。与批注、便签是两套逻辑。

2. **批注与便签**  
   列出**当前书籍**下的批注与未关闭便签；可在此删除。底部 **打开跨书关联页…** 与书籍库顶栏的「跨书关联」相同，进入全屏列表。

## 跨书关联页

- **书籍库** 顶栏按钮 **跨书关联**：从首页进入。  
- **写作页** 侧栏「关联」底部 **打开跨书关联页…**：以浮层打开，关闭后会刷新侧栏数据。  

页内展示所有勾选了「跨书」的批注与便签，可按条删除。

## 提示

- 批注依赖**选区位置**；大段改写正文后，高亮位置可能偏移，可删了重加。  
- 「跨书」适合需要在多本书之间统一查看的备忘；不需要跨书的不要勾选，列表更干净。
